# ChatVerse

ChatVerse is a multilingual chat application that enables users to communicate seamlessly across different languages.

## Features

- Real-time messaging
- Multilingual communication
- Responsive user interface
- Client-server architecture

## Tech Stack

### Frontend
- React

### Backend
- Node.js
- Express

### Database
- MongoDB
